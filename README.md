# Анонимный опрос юридической команды

Статическая страница без зависимостей. Ответы отправляются в существующую Google-форму, поэтому отдельная база данных не требуется.

## Локальный просмотр

Запустите любой статический веб-сервер в этой папке и откройте `index.html` через него.

## Важное перед публикацией

- Период опроса и тексты вопросов находятся в `index.html`.
- Состав сотрудников, функции и соответствие полям Google Forms находятся в `app.js`.
- Не отправляйте тестовый ответ в рабочую форму без согласования: он попадёт в таблицу результатов.
